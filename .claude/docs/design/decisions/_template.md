# 주제 — 한 줄 부제

## 지금의 결정

지켜야 할 규칙을 명령형으로. **여기가 이 주제의 단일 출처다.**

값은 복제하지 않고 계약 문서(`panda.config.ts` · `window-chrome.md` 등)를 가리킨다.

## 왜 이렇게 정했나

결정을 떠받치는 계산·측정. 수치가 없으면 눈으로 골랐다고 밝힌다.

기각한 대안을 나열하지 않는다. 기각 근거가 결정을 설명하는 데 필요하면 본문에 녹인다.

## 트레이드오프

- 무엇을 포기했는가.
- 게이트·테스트가 이 결정을 어떻게 강제하는가.
- 따라오는 후속 작업.
