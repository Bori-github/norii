# NNNN. 결정을 한 문장으로

- **상태**: 확정 | 대체됨 (→ NNNN)
- **날짜**: YYYY-MM-DD

## 맥락

무엇이 문제였나. 결정이 필요해진 계기.

## 고려한 대안

- **안 A** — 무엇인가. 왜 기각/채택했는가.
- **안 B** — 무엇인가. 왜 기각/채택했는가.

## 결정

무엇으로 정했는가. 지켜야 할 규칙을 명령형 한두 문장으로.

## 결과

- 좋은 점 — 이 결정이 사게 된 것.
- 나쁜 점 — 이 결정이 치르는 비용. **숨기지 않는다.**
- 따라오는 일 — 이 결정 때문에 해야 하는 후속 작업.
